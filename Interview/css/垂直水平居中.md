# 盒子垂直水平居中

```css
/* 方法一 absolute transform/margin */
div {
  position: absolute;
  width: 500px;
  height: 300px;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  background-color: pink;
}

/** 方法二 flex **/
.container {
  display: flex;
  align-items: center;        /* 垂直居中 */
  justify-content: center;    /* 水平居中 */
}

.container div {
  width: 100px;
  height: 100px;
  background-color: pink;
}
```
